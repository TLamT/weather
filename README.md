### 📋 功能一覽

| | |
|---|---|
| 🌡️ **即時溫度** | 顯示香港天文台及香港平均溫度 |
| ⚠️ **天氣警告** | 使用天文台官方圖示，各種警告一目了然 |
| 📍 **地區溫度** | 切換各區即時氣溫 |
| 📅 **九天預報** | 未來九天天氣預報 + 溫度範圍圖 |
| 🕒 **實時時鐘** | 當前日期時間顯示 |
| 📄 **A4 排版** | 文件式美觀佈局 |
